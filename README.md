# Mebar Solar

React + TypeScript starter based on the local `mystique_realm` project setup.

## Scripts

- `npm start` runs the development server.
- `npm run build` creates a production build.
- `npm test` starts the CRA test runner.

## Global Site Template

Edit `src/config/site.ts` for site-wide labels, routes, contact details, and page copy.
Edit `src/styles/theme.css` for global CSS variables such as colors, spacing, and font sizes.
